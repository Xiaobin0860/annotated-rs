# substrate 阅读记录:

## 阅读方法:

- 有示例, 从示例入口开始阅读. 没有, 查看 test 代码.
- 通过示例方式, 拆解框架.
- 这是最高效的阅读方式

> 示例入口:

- [examples](./tokio-1.14.1/examples)
