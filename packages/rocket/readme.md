# Rocket 阅读记录:

## 阅读方法:

- 从示例入手, 逐步拆解整个 web 框架
- 这是最高效的阅读方式

> 示例入口:

- [examples](./rocket-0.4.10/examples)

## 默认注解规则:

- 所有新增注解, 注释前缀为如下2种格式:
- 区分原 repo 注释, 以及方便醒目查看

```ruby 

// TODO X: xxx
// todo x: xxx

```

- `jump in`标志, 此处可以跳转到 web 框架, 会继续注解
